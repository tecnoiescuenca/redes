Configuración y Simulación de Redes Informáticas

¿Qué vamos a conseguir?

Nuestros Objetivos y el Gran Reto.

En esta unidad, dejarás de ser un usuario pasivo para convertirte en un diseñador o diseñadora de redes informáticas, siendo capaz de configurar la red de internet de tu casa. Al completar este reto, habrás logrado:

  • Identificar los elementos físicos que forman una red (router, cables, puntos de acceso) y para qué sirve cada uno.
  • Diseñar y simular una red doméstica profesional utilizando el software Cisco Packet Tracer.
  • Tomar decisiones de seguridad reales, como configurar protocolos de cifrado y gestionar contraseñas para proteger tus datos.
  • Crear y compartir tu propio conocimiento mediante infografías y vídeo-tutoriales, para ayudar a que otras personas también naveguen seguras.
  • Conectar con el Futuro (IoT): Comprenderás cómo la red se expande más allá de los ordenadores, analizando el impacto del Internet de las Cosas.
